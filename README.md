# TimeLogD (가제, 이하 TLD)

일기장, 인데 타임라인에 초점을 맞춘 기록 앱.

기존의 캘린더가 단순 일정 관리를 위한 앱이라면, TLD는 사용자의 **하루를 시간 단위로 나눠 기록하는 앱**입니다.

<br><br>

## 2023-06-20
### 팀 개설 및 주요 기능 설계

- 월간/주간/일간 캘린더
- 일간의 경우, 타임라인 형식으로 제공되어 시간 단위로 일과를 기록
- 각 시간 별 블로그 형식의 글 작성 및 저장 (사진 첨부 가능)
